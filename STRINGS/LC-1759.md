# Count Number of Homogenous Substrings

```cpp

class Solution {
public:
    int countHomogenous(string s) {
        int n = s.size();

        int result = 0;
        int length = 0;

        for(int i=0;i<n;i++) {

            if(i > 0 && s[i] == s[i-1]) {
                length++;
            }
            else {
                length = 1;
            }
            result += length;

        }

        return result;
    }
};

```
